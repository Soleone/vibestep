# Third-party media tools

Vibestep Companion downloads the following tools on first launch. They are not authored by Vibestep and are stored in the user's private companion data directory. Exact versions, binary URLs, and SHA-256 checksums are defined in `companion/tools.js`.

## yt-dlp

- Project: https://github.com/yt-dlp/yt-dlp
- Source and release materials: https://github.com/yt-dlp/yt-dlp/releases
- License: The Unlicense, with separately licensed bundled components documented by the project

## FFmpeg and ffprobe builds

- Binary distributor: https://github.com/shaka-project/static-ffmpeg-binaries
- FFmpeg source: https://ffmpeg.org/download.html
- Build source and release materials: https://github.com/shaka-project/static-ffmpeg-binaries/releases
- License: GPL-licensed build components may apply. See the distributor's release and license materials for the exact pinned build.

These notices do not replace the upstream license texts or any source-distribution obligations. Release owners must verify the notices and corresponding-source availability for each pinned version before publishing a companion installer.
